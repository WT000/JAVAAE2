Index Page to access App

The web app opens on the home page

DEFAULT_ADMIN_USERNAME = "globaladmin"

DEFAULT_ADMIN_PASSWORD = "globaladmin"

DEFAULT_USER_PASSWORD = "user1234"

DEFAULT_USER_USERNAME = "user1234"

Logs are kept in (TomcatDir)/logs/ae2cart

Properties are kept in (TomcatDir)/temp/application.properties (it'll automatically generate if no properties file exists, else it'll load what's in there)

The navigation bar is based on the template bootstrap-starter.html

Which is documented here https://getbootstrap.com/docs/3.3/examples/navbar/
